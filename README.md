# E-commerce Platform

Monorepo for an e-commerce application built with Spring Boot (Hexagonal Architecture) and Angular.

## Structure
- **backend/** – Spring Boot 3 + Java 21 using Hexagonal Architecture.
- **frontend/** – Angular 20 application (to be added).

## Roadmap
- PayPal integration for payments
- Security module and authentication with JWT
- Domain-driven improvements and modularization
